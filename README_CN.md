## 简介
Go-With-STL是基于 `golang` 泛型实现的数据结构与算法的库，不完全等同于 `C++` STL。老实说，它并不是一个真的STL，甚至连tiny都不是。由于泛型无法比较和反射等原因，我没有找到一个好的方式去分离容器和迭代器。可能在将来会实现它，谁知道呢？ XD
